Identificando JS em paginas da web

Acima, vemos a pagina de Login de uma das redes sociais mais utilizadas no mundo, a plataforma X, atigo twitter, comprado em Abril de 2022 pelo sul africano, Elon Musk

Até em grandes redes sociais, vemos a presença da linguagem Java Script, nesse caso acima, o JS é usado para validar o formulario onde o usuario insere seu uma das tres informações solicitados, alem de buscar em um provavel banco de dados, se aquele usuario tem ou não uma conta no X Alem dos botões de acesso onde são enviadas essas informações para o servidor. Tambem vemos que ele busca em outros servidores (gmail e apple) informações a respeito de contas presentes no x.

Extensões uteis para desenvolvimento WEB

Auto Close Tag:

Essa tag é interessante pois ela fecha as tags em html, o que otimiza o tempo e diminui probalidade de precisar rodar todo o codigo pra identificar um erro

HTML CSS Support:

Uma tag interessante para pessoas que não dominam o ingles ou são iniciantes em desenvolvimento web, ela sugere sugestão de codigos em CSS

CSS Lint

Essa tag auxilia na correção do codigo, uma vez que ele percorre o codigo e aponta provaveis erros

ESlint:

Essa é uma boa tag para aprendizes em JS, ela ajuda na construção do codigo, identificante provaveis erros de sintaxe, padrões inconsistentes e alguns outros problemas, alem de ser super personalizavel permitindo que o dev tenha mais autonomia
