Identificando JS em paginas da web

Acima, vemos a pagina de Login de uma das redes sociais mais utilizadas no mundo, a plataforma X, atigo twitter, comprado em Abril de 2022

Até em grandes redes sociais, vemos a presença da linguagem Java Script, nesse caso acima, o JS é usado para validar o formulario onde o usuario insere seu uma das tres informações solicitados, alem de buscar em um provavel banco de dados, se aquele usuario tem ou não uma conta no X Alem dos botões de acesso onde são enviadas essas informações para o servidor. Tambem vemos que ele busca em outros servidores (gmail e apple) informações a respeito de contas presentes no x.
